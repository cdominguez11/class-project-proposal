▶ Open LockedIn App

  1. Product Name: LockedIn
  2. Team Members: Chris
  3. Application Overview: LockedIn is a web-based focus app that lets you set a countdown timer and manage a to-do list
  4. Technology Overview: HTML, CSS, and JavaScript. Google Fonts for typography. Web Notifications API for session complete alerts. Can be run as a local HTML file
  5. Proposal Last Updated: 05/11/2026
  6. Feature List:
    • Editable countdown timer
    • Bullet point list
    • Light/dark mode
    • Keyboard shortcuts to control the timer (Enter to start, Space to pause)
    • Session complete notifications
  7. Future Feature List:
    • Sound Alerts
    • Session history to track completed focus sessions
    • Custom Themes
    • Menu bar widget for Mac
  8. Basic Technical Features:
    • Used HTML to build the structure of the page including the timer, task list, and layout
    • Used CSS to style the app with custom fonts, colors, and spacing
    • Used CSS Grid to organize the timer and task list on the screen
    • Used JavaScript variables to keep track of the timer and whether it is running or paused
    • Used JavaScript functions to start, stop, and reset the timer
  9. Advanced Technical Features:
    • Used a CSS media query to automatically switch between light and dark mode based on the device settings
    • Used CSS animations to make the timer pulse when time is almost up and flash when it hits zero
    • Used the Web Notifications API to send a browser notification when the session is done
    • Used Date.now() to track real clock time so the timer stays accurate even when the tab is in the background
    • Used contentEditable to let users click on a task and edit it directly without any extra input boxes
    • Built a keyboard shortcut system so Enter starts the timer, Space pauses it, and Escape cancels without breaking anything while typing a task
